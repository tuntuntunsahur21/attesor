## 🔧 How to Use attesor

This section covers basic steps to use attesor and understand its main functions.

1. **Load a Program:** Click “Open” or “Load” in attesor to select a file you want to analyze. This file should be a Linux program you want to reverse-engineer.

2. **Start Analysis:** press the “Analyze” button. The AI will work to understand the program’s structure. This may take a few minutes.

3. **View Results:** After the analysis finishes, attesor displays a report. This report shows parts of the program and hints about how it works.

4. **Explore Details:** You can click sections of the report to get deeper info. This helps if you want to learn more about specific parts of the software.
